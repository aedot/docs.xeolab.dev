## Set date on Linux using timedatectl

- Show current timezone
```
timedatectl
```
- List timezones
```
timedatectl list-timezones
```
- Set timezone
```
timedatectl set-timezone America/New_York
```
- enable time synchronization
```
timedatectl set-ntp true
```
